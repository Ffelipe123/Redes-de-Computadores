  Neste laboratório no Cisco Packet Tracer, foi configurado o acesso remoto 
a um roteador utilizando SSH versão 2 (SSHv2), com criptografia RSA de
2048 bits, em uma rede IPv4. No roteador remoto, foram feitas as
configurações necessárias para habilitar o SSH, como definição de
domínio, criação de usuário local e liberação do acesso nas linhas VTY
apenas via SSH, garantindo mais segurança no gerenciamento do equipamento.

  Além disso, também foram realizadas configurações básicas nos dispositivos
intermediários, como hostname e banner de aviso, para melhor identificação
e organização da rede. Para que o host conseguisse se comunicar com o 
roteador remoto, foi utilizado roteamento estático em IPv4, permitindo o 
encaminhamento correto dos pacotes. Por fim, foram feitos testes para 
verificar a conectividade e confirmar que o acesso remoto estava funcionando
corretamente
